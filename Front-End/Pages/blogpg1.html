What Is JSON? Explained With JSON Examples

What is JSON?

What is JSON? JSON is a way to store, persist data in a structure which is easy to read and easy to access. JSON stands for JavaScript Object Notation, which machine can understand and generate. JSON structure provides a human-readable collection of information to explain and understand the logical representation of data. Let's get to the JSON structure.

  1. {
  2. "Actors": [
  3. {
  4. "id": "1",
  5. "firstName": "Tom",
  6. "lastName": "Cruise"
  7. },
  8. {
  9. "id": "2",
  10. "firstName": "James",
  11. "lastName": "Bond"
  12. }
  13. ]
  14. ]

This example shows the Actors' data and has two actors. It's easy to understand along with that it takes less space to store in memory than XML.

Above JSON data consists of an array of Actors and each actor is value in JSON Array. Looks complicated? Let's understand all the data type JSON has, to represent the data.

JSON Data Types

  • Number
  • String
  • Boolean
  • Array
  • Object
  • Null

This example uses all the JSON data types.
