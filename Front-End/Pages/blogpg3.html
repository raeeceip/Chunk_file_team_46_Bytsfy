JSON Example With Data Types Including JSON Array

JSON Example This article will have all the JSON Examples which covers each and every data type JSON supports. Here is the list of JSON data types. Valid JSON Data Types

  1. String
  2. Number
  3. Object
  4. Array
  5. Boolean
  6. Null
  1. JSON String Example:
    1.     {
    2.       "firstname": "Tom",
    3.       "lastname": "Cruise",
    4.       "occupation": "Actor"
    5.    }

    This example shows information about a person, and you know Tom Cruise. This JSON data represent details which contain the string. Play with JSON String Example. String in JSON has to be in double-quotes.

  2. JSON Number Example:
    1.     {
    2.       "id": 1,
    3.        "age": 56,
    4.       "bornyear": 1963,
    5.       "date": 3,
    6.       "month": 7,
    7.       "weight" : 67.5
    8.    }

    This example shows how to represent the numbers and I have added the birthday details of Tom Cruise. I have added this in 2018. So you can do the math if this is not 2018. ðŸ™‚ Play with JSON Number Example. Numbers in JSON has to be an integer or a floating-point.

  3. JSON Object Example:
    1.     {
    2.          "actor": {
    3.           "name": "Tom Cruise",
    4.          "age": 56,
    5.          "Born At": "Syracuse, NY",
    6.          "Birthdate": "July 3 1962",
    7.          "photo": "https://jsonformatter.org/img/tom-cruise.jpg"
    8.       }
    9.    }

    This JSON data represent the detail about the actor and its property. It is a JSON object which can have different properties. Play with JSON Object Example.

  4. JSON Array Example
    1.     {
    2.          "Actors": [
    3.              {
    4.                "name": "Tom Cruise",
    5.                "age": 56,
    6.                "Born At": "Syracuse, NY",
    7.                "Birthdate": "July 3 1962",
    8.                "photo": "https://jsonformatter.org/img/tom-cruise.jpg"
    9.                   },
    10.              {
    11.                "name": "Robert Downey Jr.",
    12.                "age": 53,
    13.                "Born At": "New York City, NY",
    14.                "Birthdate": "April 4, 1965",
    15.                "photo": "https://jsonformatter.org/img/Robert-Downey-Jr.jpg"
    16.                   }
    17.               ]
    18.    }

    This is an Array of JSON data. It shows two actor objects and each object can have its own properties. Play with JSON Array Example.

  5. JSON Boolean Example
    1.     {
    2.          "hasChildren":true,
    3.           "hasTwitterAccount":true,
    4.          "hasGreyHair":false
    5.    }

    This JSON Example shows how to represent Boolean values in JSON. Play with JSON Boolean Example. Boolean value has to be true or false.

  6. JSON Null Example
    1.     {
    2.          "wife":null
    3.    }

    JSON Null Example shows how to represent Null values in JSON. Play with JSON Null Example. Null will help to represent value as not available.
